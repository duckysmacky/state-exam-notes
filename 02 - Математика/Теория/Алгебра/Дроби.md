## Понятия

- Числитель (делимое) - число сверху
- Знаменатель (делитель) - число снизу

## Правила работы с дробями

1. (a + b) / c = a / c + b / c
	a / (a + c) != a / b + a / c
2. a / b + c / d = (a * d + c * b) / (b * d)
3. (a / b) * (c / d) = (a * c) / (b * d)
4. (a / b) / (c / d) = (a / b) * (d / c) = (a * d) / (b * c)
5. a * (b / c) = (a * b) / c
